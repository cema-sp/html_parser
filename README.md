# HTML parser

Simple API to fetch URL and parse H1, H3, H3 and A tags.

## Stack

* Ruby 2.3.3
* Rails 5.0
* Puma

## ENV vars

## Installation

1. Clone repo
2. Prepare DB & set ENV vars (see [ENV vars](#env-vars))
3. `bundle install`
4. `rails db:migrate`
5. Run test suite: `rails t`
6. Start server: `rails s`
